### 🚀 What I'm building

**Sentinela** — Full-stack SaaS that monitors Brazil's *Diário Oficial* (official gazette) for lawyers, in collaboration with one teammate. I own the entire backend.

- Layered architecture (controller → service → repository → DTO) with Spring Boot 3
- Authentication & authorization with Spring Security 6, JWT and OAuth2
- RESTful APIs documented with Springdoc OpenAPI 2 (Swagger UI)
- PostgreSQL with version-controlled schema migrations via Flyway
- Containerization with Docker and Docker Compose
- Global exception handling and structured error responses

**Stack:** Java 21 · Spring Boot 3 · Spring Security 6 · Spring Data JPA · PostgreSQL · Flyway · Docker · Maven · JWT · OAuth2 · OpenAPI/Swagger
